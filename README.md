# Seb's 

type words. create jazz.

this is a small frontend project built with html, css, js, and tone.js that turns typing into music! 

every letter triggers musical notes, and your writing can be transformed into a generated performance. the goal wasn't to build a perfect music generator, but to explore how text, sound, and interaction can come together in a playful way ;)

built with: 

* html
* css
* js
* tone,js

what i learned: 

this project taught me:

* how browser audio works
* using Tone.js for sound synthesis
* event listeners and keyboard interactions
* mapping user input to musical notes
* structuring a basic frontend project
* working with Git and GitHub
* deploying projects using gitHub pages
* how to turn a simple idea into a working project
* the importance of experimenting and learning through trial and error. 

created to bring a little bit of jazz into the browser <3
